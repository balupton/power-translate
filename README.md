# power-translate

Translates text from a language to all available languages

``` bash
npm install -g power-translate
power-translate "$GOOGLE_CLOUD_TRANSLATE_API_KEY" "$FROM_LANGUAGE" "$FROM_TEXT"
```

e.g. this command:

``` bash
power-translate "$GOOGLE_CLOUD_TRANSLATE_API_KEY" en "Hello World"
```
